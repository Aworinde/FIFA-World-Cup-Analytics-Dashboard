# FIFA-World-Cup-Analytics-Dashboard

## Overview
This project showcases an interactive **FIFA World Cup Analytics Dashboard** built using **Power BI**. It provides a comprehensive analysis of historical World Cup data, including matches, goals, teams, spectators, and performance trends over time.

The dashboard transforms raw football data into meaningful insights to support storytelling, trend analysis, and decision-making—demonstrating strong **data cleaning, modeling, and visualization skills**.

---

## Project Objectives
- Analyze historical FIFA World Cup data across multiple years  
- Identify trends in matches, goals, and spectators  
- Evaluate team participation and tournament structure  
- Compare performance metrics such as home vs away goals  
- Present insights through interactive and visually appealing dashboards  

---

## Tools & Technologies
- **Power BI** – Dashboard development and visualization  
- **Power Query** – Data cleaning and transformation (ETL)  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **Excel / CSV** – Data source  

---

## Dataset Description
The dataset includes:
- Match statistics (games, goals, match outcomes)  
- Tournament details (years, host countries, teams)  
- Stadium and venue information  
- Spectator data  
- Match scheduling (days, stages, rounds)  

---

## Key Dashboard Features

### 1. World Cup Overview
- Total Matches: **836**  
- Total Goals: **2,379**  
- Total Teams: **83**  
- Total Stadiums: **181**  
- Analysis period: **20 tournaments**  

---

### 2. Spectators Trend Analysis
- Historical trend of spectators over time  
- Forecasting future attendance patterns  
- Clear upward growth trend in global audience engagement  

---

### 3. Matches by Weekday
- Highest matches played on:
  - Saturday (**132K**)  
  - Sunday (**126K**)  
- Lowest activity:
  - Thursday & Monday  

Insight: Weekend scheduling maximizes audience engagement.

---

### 4. Stadium & Tournament Growth
- Increase in stadium usage over time  
- Expansion of tournament scale  
- Modern tournaments show higher infrastructure investment  

---

### 5. Participating Countries (Geospatial Analysis)
- Global distribution of participating teams  
- Strong representation across:
  - Europe  
  - South America  
  - Africa  

---

### 6. Quadriennial Analysis (Tournament-Level Insights)
- Year-by-year tournament breakdown  
- Host country, winners, and match statistics  
- Matches played trend increasing steadily over time  

---

### 7. Home vs Away Goals Analysis
- Comparison of home goals (HG) vs away goals (AG)  
- Balanced scoring trends across tournament stages  
- Slight dominance in specific match stages  

---

### 8. Tournament Stage Performance
- Goal distribution across:
  - Group stages  
  - Knockout rounds  
  - Finals  
- Highest scoring observed in early-stage matches

## Data Cleaning & Transformation
Performed using **Power Query**:
- Removed duplicates and inconsistencies  
- Standardized naming conventions (countries, stages)  
- Handled missing/null values  
- Created derived columns for time-based analysis
